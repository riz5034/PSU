About Us It's Nice to Meet You!

The Vinyl cutting team has gathered together to built a working site for QAD. To do so we shall use the Amazon web service to build a site using the tools it possesses such as the LambdaTM, DynamoDBTM, and CognitoTM. Each of them filling in for a necessary position in the construction of the site, from the login to the orders. Using our lessons from school we've built a site that fulfills all that the customers and employers will need while still retaining its flexibility to grow larger if need be. It started with building a site and then having it store data valuables and accounts, with those done all we needed was to build the site around them. Soon we expanded into building the other pages. Along the way, we had to solve issues with minor things such as the lost password function and getting the proper feedback from the various tools. We persevered and built a site we can be proud of.

With this site, we made it so one can order various products like stickers, decals, or small messages that can be placed where-ever one wishes to. these professionally cut designs will look impressive and project a feeling of professional reliability. The site is also the best it can get in terms of like-ability as it is fast and streamlined in terms of use and check out. After all, nobody wants to be stuck searching a site for what you want.

Our Team

Chan Morn Team Leader, Web Designer

Chan is the team leader of Team Vinyl Cutting. Her job is to lead the team to achieve their goal of all end to end web application built ERP that allows customers to purchase Vinyl Cutting equipment and handle the order processing of purchases. Chan also is the head of mockups and the CSS designs for the interface on both the customer and admin ERP side of the application.

Ricky Zhao Front and Back End Developer

Ricky is the head developer on the customer and admin side of the ERP application. His job is test develop the functionality through Javascript and other necessary web tools, to achieve a goal of an end to end functioning ERP Vinyl Cutting application.

Zachary Sigamony Middleware and Back End Developer

Zachary is the developer for the midlleware and back end functionality of the ERP application, as well as a developer for the admin side of the ERP application. His job is to create, test and deploy middleware API Gateway's with AWS, to pass through payloads to save to the Dynamo database for order processing.

Michael Morrell Middleware and Back End Developer

Michael is the developer for the midlleware and back end functionality of the ERP application, as well as a developer for the admin side of the ERP application. James O'Connell QA Analyst, Lead Tester
